University Management System (TypeScript)

Цей проєкт реалізує систему управління університетом на TypeScript.

Можливості:

- Додавання студентів
- Додавання курсів
- Реєстрація студентів на курси
- Виставлення оцінок
- Зміна статусу студента з валідацією
- Отримання списку студентів факультету
- Обчислення середнього балу
- Отримання списку відмінників

Увесь код розміщений в одному файлі: `/src/university.ts`.
